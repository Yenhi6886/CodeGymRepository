Đây là tiêu đề thứ nhất

Connector/NET supports the calling of stored procedures through the MySqlCommand object. Data can be passed in and out of a MySQL stored procedure through use of the MySqlCommand.Parameters collection.

Cái này là tiêu đề thứ hai

After the stored procedure is named, you define one MySqlCommand parameter for every parameter in the stored procedure.

Còn này là tiêu đề thứ ba

A stored procedure is a set of SQL statements that is stored in the server
